INTRODUCCIÓN
A continuación, se presenta la guía de la aplicación web de Lucade System,
donde se explica todas y cada una de las funciones que contiene el sistema,
Este manual de usuario está dividido en varios capítulos donde se explica los
procesos, y como funciona cada uno de ellos.
Esta aplicación web esta exclusivamente desarrollada, Para generar procesos
de facturación, se recomienda que se lea este manual antes de iniciar a utilizar
esta aplicación, para así tener un mejor manejo y rendimiento de todas las
opciones que posee este programa.
Además, cuenta con un glosario donde se explica el concepto de cada una de
las palabras que no se entiendan y que se considera que es necesario tener el
concepto claro, para así tener un mejor uso de la aplicación.
Se recomienda que cada administrador que utilice esta aplicación debe estar
obligado a leer este manual, para evitar problemas en un futuro.

TIPOS DE USUARIOS:
-Gerente
El gerente tendrá acceso a todos los módulos del sistema y configuraciones con
privilegios de administrador.
-Vendedor
En este rol de empleado este tendrá los privilegios de poder acceder a cuatro
módulos del sistema que son clientes, ventas, inventarios y traslados que estos
se detallaran más adelante.
-Compras
Este rol tendrá acceso solo a un módulo llamado compras el cual se detallará
detalladamente más adelante.
-Bodegero
Este rol tendrá privilegios de acceder a los siguientes módulos de compra,
inventario y traslados.
Estos roles serán los usuarios de acceso al sistema que serán capaces de
realizar los procesos que el software va a facilitarles para que desarrollen sus
actividades optimizadas.
